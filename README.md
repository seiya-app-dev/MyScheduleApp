# MyScheduleApp

## 概要

C#（WinForms）とSQL Serverを使用したスケジュール管理アプリです。  
ユーザーごとに予定を登録・編集・削除でき、他ユーザーの予定も閲覧・操作可能です。

## 主な機能

* ログイン/ログアウト機能（SHA-256によるパスワードハッシュ化）
* スケジュール登録・削除・復元
* 管理者権限によるユーザー管理
* 論理削除・復元機能（IsDeletedフラグ）
* ログ出力機能（操作履歴の記録）

## 開発環境

* **言語**：C#
* **フレームワーク**：.NET Framework 8.0
* **DB**：SQL Server Management Studio
* **開発環境**：Visual Studio 2022
* **バージョン管理**：Git / GitHub
